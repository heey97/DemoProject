1. 파일 업로드를 실행할 창 구현 (bookcaseFrom.jsp)
    ㄴ 필요한 dto의 객체에 맞게 입력받은 파라미터를 가져올 input 태그를 생성.
    ㄴ 파일 업로드를 위해 input 상위의  form 태그에 enctype="multipart/form-data" 추가
    ㄴ file type인 input 태그에 image 파일을 가져올 것 이므로  모든 이미지 확장자를 포함하기 위해 accept="image/*"  추가
    ㄴ 버튼 태그의 타입을 submit으로하여 버튼을 눌렀을 때  
        form action url인 "/book/save"와 method POST로 지정해놓은 BookCaseSaveController로  파라미터를 전송 

2. 파일 업로드를 위한 VIEW(JSP)에서 데이터를 입력받고 전달 해주기위한 Controller 구현
    ㄴ 파일을 가져올 경로 및 가져올 파일의 최대 용량 설정 ex) String path = "D:\\Class231228\\upload",  
					                  int maxSize = 10*1024*1024 (1024바이트 = 1KB, 10MByte로 설정)
    ㄴ 뷰로부터 파일 타입 파라미터를 전달받기위해  (servlet 라이브러리중 하나인 cos 라이브러리 추가가 필요합니다.)
         MultipartRequest를 사용하여 전달방법, 경로, maxsize, 인코딩, DefaultFileRenamePolicy객체 정의
         (DefaultFileRenamePolicy는 중복된 파일명에 대해 새로파일명 부여하는 규칙(파일명 뒤에 일련번호))
    
    ㄴ 정의한 객체의 getparameter 메소드를 사용해 해당하는 파라미터값 전달받음
    ㄴ 서버로 전송해 저장할 파일의 이름 설정
    ㄴ 데이터를 전송하기 전 권한 비교하여 등록이 실패하면 메시지 전달
